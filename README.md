# Pong game 🏓
`Python 3.11.1`

Play classic pong game and be the best 🕹️

## How it works
#### Left paddle
- Use the `W` key to go up
- Use the `S` key to go down
#### Right paddle
- Use the arrow key `⬆️` to go up
- Use the arrow key `⬇️` to go down
